Colophon

This site is built using several tools.

  • The core of this site is built with 11ty and hosted on Netlify.
  • The content for my media pages are stored in a Sqlite database and managed with my Media Repository running on my Synology NAS. The downside is that I can only build the media pages from home as the repository is local, but it doesn't cost me anything and I do almost all of my work from home.
  • I go back and forth with my writing between iA Writer and Obsidian. Regardless of app, when the posts are ready, I copy them into my 11ty directory and they're built there.
  • I use Namecheap as my domain registrar.
